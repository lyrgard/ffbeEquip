Icons can be sized via .icon .icon-* (xxs, xs, sm, md, lg, xl, xxl)

Any icon can now have a stack via .icon .icon-stack .stack-*. No wrappers, no additional code required. Current stack options are: stack-shield, stack-sword, stack-armor, stack-rod, stack-staff, stack-exclusive

Default stacking position is bottom right, but can be adjusted to any corner:

For a complete list of available icons review: /assets/css/components/icons
