How to use this:

  • 1Items that are missing data are displayed on the right. You can select one, or use the search if you want to correct another item.
  • 2Once you've found the item you want to modify, click on the to add this item to the list on the left. You can add multiple items to this list (max 10).
  • 3You can modify the Access and Max Number of the items on the left pane. For Max Number, leave it empty if there is no limitation to how many times this item could be acquired.
  • 4When you have entered all the data to the items, click on Send it to FFBE Equip to send it to FFBE Equip.
  • 5A dialog should appears with the result of your submission.
Please note that the data won't be corrected immediately. I regularly merge the sent data to the main data of FFBE Equip.
