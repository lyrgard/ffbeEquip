Advanced Usage

More options
  • If you plan to use an attack with innate element(s) or your unit has been imbued with an element, you can select the corresponding element(s) in the "Skill innate element(s)" section (for instance, you'll want to select Ice if you plan to use Fryevia's Frost Flower Blitz.
  • Fill information on the monster you're facing, its race(s) and its elemental resistance (negative number for elemental weakness).
  • Take into account the elemental debufs you'll give it (for instance, if using Orlandeau's Divine Ruination, substract 50 from the light elemental resistance).
  • You can adjust the DEF/SPR of the monster to account for breaks.
  • Custom Formula allows to create tailored-made builds. More information on how to use it when you click on it.
  • Some pre-set custom formula can be found on the custom formula popup. Click on the arrow on the left of the text field to get access to various healing and expedition custom formula.
Equipment Selection
All Equipment
  • All available Equipment, unless limited by available options.
  • Espers at Maximum rarity and level, with only Killers unlocked.
Only Owned Equipments
  • Only available when logged in.
  • All equipments inputed in the inventory page.
  • If espers are inputed in the Espers page, will use that, else default to the same espers as All Equipments.
Only Owned Equipments Available to Expedition
  • Only available when logged in.
  • Same as Only owned equipment, but exlude items marked as excluded from expedition in the inventory page.
  • No esper will be automatically selected.
  • For ease of use, pre-set custom formula can be found on the custom formula popup. Click on the arrow on the left of the text field to get access to the expedition custom formula
Only Unlimited Equipments (not event shop/recipe)
  • Only items available at will from shop/recipe and not from event will be used.
  • No esper will be automatically selected.

Why Log In?

To make builds tailored just for your available equipments
  • Log in and go to the Inventory page.
  • Take the time to input your inventory, and update it as necessary.
To allow this tool to suggest you TMR to farm
  • Log in and go to the Units page.
  • Take the time to input your units, and adjust the available TMRs.
  • When building with "Only owned equipments", select "Include TMR of owned units not farmed yet"
To be able to use your own esper builds and not the default ones
  • Log in and go to the Espers page.
  • Take the time to input your esper information.
  • Note: The Espers page uses "path finding." This allows you to select objects on the outer-most ring, and claim all objects along that path. For example, Selecting Ifrit's 75% Beast Killer selects a total of 9 objects, starting with Fire, for a total of 370 SP.
  • Note: To remove an added object, simply click on it again. Path finding works here as well, allowing entire sets of objects to be removed. For example, Ifrit's Beast killer can be trimmed back to the 50% killer by clicking on the ATK + 100 object that follows the 50% killer.
